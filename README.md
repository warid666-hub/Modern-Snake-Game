# Modern Snake Game

A modern Snake game built with Python and Pygame featuring smooth movement, neon-style graphics, and a responsive control system.

## Features
- Smooth snake movement (not grid-choppy)
- Snake growth with food collection
- Collision detection with walls and self
- Score system
- Clean, modern UI with neon colors
